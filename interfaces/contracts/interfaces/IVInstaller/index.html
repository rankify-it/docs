Solidity API#

VersionRequirementTypes#

Enum defining the types of version requirements for repositories. - All: Matches any version. - MajorVersion: Matches any version with the same major version number. - ExactVersion: Matches the exact version specified.

enum VersionRequirementTypes {
  All,
  MajorVersion,
  ExactVersion
}
